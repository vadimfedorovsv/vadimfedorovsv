# Vadim Fedorov

Electrical Engineering student at UC Davis focused on robotics, embedded systems, PCB design, ROS2/micro-ROS, and BCI/EEG interfaces.

I work on real hardware systems where firmware, electronics, and robotics software meet. Current work includes embedded firmware and electronics debugging for Krysalis Hand, a 19-DOF humanoid robotic hand research platform at the Laboratory for AI, Robotics and Automation (LARA).

## Current Focus

- Embedded firmware for robotics hardware
- PCB-level debugging and hardware bring-up
- ROS2/micro-ROS pipelines on physical systems
- EEG/BCI signal processing and control interfaces
- Low-latency communication across distributed embedded components

## Featured Work

### Krysalis Hand

Embedded firmware and electronics debugging for a 19-DOF humanoid robotic hand research platform.

Highlights:
- Diagnosed I2C bus contention across distributed embedded components
- Reworked inbound/outbound signal paths for more reliable command and sensor data flow
- Contributed to PCB-level improvements including pull-up resistor selection and voltage regulation
- Integrated embedded firmware work with ROS2/micro-ROS robotics pipelines

### PyBCI EEG Research Platform

Python-based EEG/BCI research tooling for signal acquisition, preprocessing, feature extraction, visualization, and control-output prototyping.

### BCI Prosthetic Leg

Prototype control pipeline connecting EEG/BCI signal features to assistive robotic motion commands.
